微软Exchange客户端服务器信息泄露

Kali-Team
2018-05-25
web

关于

可以泄露远程Microsoft Exchange环境的内部IP地址。这包括承载诸如Outlook Web App（OWA）和自动发现等服务的客户端访问服务器（CAS）的内部地址。这还包括代理或网关处理OWA请求的内部地址。
