我的Linux软件目录

Kali-Team 发表于 2018-09-13

远程桌面类的

  • 远程桌面查看器，vinagre：支持多种协议
  • Remmina：这个也不错
  • TeamView：有时会用到

终端

  • Tilix：不知道为什么选这个终端，因为可以设置背景图片。
  • zsh：加上Oh-My-Zsh

效率工具

  • 主菜单，alacarte：一个可以创建快捷方式的工具
  • Evolution：收发邮件，日程表。
  • Vim：神级编辑器
  • Zeal：开发文档，可以快速搜索，支持订阅
  • albert：快捷搜索打开软件
  • stardict：可以离线翻译，可以导入字典。
  • Typora：Markdown编辑器
  • Maltego：信息收集神器
  • Xmind：脑图

系统工具

  • 清理工具，BleachBit：功能强大
  • 杀毒软件，ClamAV：开源免费还可以查杀Windows系统的威胁；ClamTK：ClamAV的GUI版本，可以安装插件添加右键扫描文件夹。
  • 输入法，搜狗输入法：符合国人输入习惯，支持云联想。
  • Wireshark-QT版：流量数据分析
  • WPS：办公软件
  • deepin-screenshot：很像QQ的截图功能，可以选定窗口截图，同时还可以编辑。
  • Nmap：神器
  • Timeshift：系统备份还原工具
  • Shadowsocks-Qt5：代理

视频影音

  • 录屏推流软件，OBS：免费开源插件多
  • 音乐播放器，网易云音乐：比较好看，但本地的我一般使用Cmus
  • VLC：免费开源插件多，可以拉流。

娱乐

  • PlayOnLinux：可以安装很多Windows上的游戏

  • Steam：一个游戏平台

聊天软件

  • qTox：P2P加密聊天软件，功能强大
  • TIM：腾讯的
  • WeChat：腾讯的
  • electronic-wechat：网页版微信

下载软件

  • Transmission：解析种子的
  • uGet：还行
  • 百度云盘：有时候会用到
  • rslsycn：P2P大文件同步工具

开发

  • Idea全套
  • anaconda：管理Python库和环境的
  • sublime-Text3：轻量级编辑器
  • vscode：这个编辑器牛逼，开源的力量。
  • OpenCV：开源计算机视觉库
  • Tensorflow：神经网络模型和深度学习框架
  • dbeaver：强大的数据库管理套件

虚拟化

  • docker：搭建环境很方便
  • VMware Workstations：商业虚拟机，没Vbox这么多BUG
  • Vbox：有时候用到

浏览器

  • Chrome
  • Firefox
  • Tor-Browser
